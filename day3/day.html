Extentions In VS Code:

VS Code extensions let you add languages, debuggers, and tools to your installation to support your development workflow. VS Code's rich extensibility model lets extension authors plug directly into the VS Code UI and contribute functionality through the same APIs used by VS Code.

Emmets and their Abbreviations:

Emmet is a powerful toolkit and abbreviation syntax used by web developers to speed up HTML and CSS coding. It allows for the shorthand entry of complex code structures, significantly improving productivity and reducing manual coding time.

Emmet Abbreviation
! Expands to the basic HTML structure, including the <!DOCTYPE>, <html>, <head>, <body>, and more.
+ used to create sibling element
& Creates a child element.
* Repeats the element a specified number of times.
^ Moves up one level in the hierarchy,allowing you to add sibling elements to parents elements
{ } Used to insert text or content inside an element.
[] Used for attribute.
( ) Groups elements for specific nesting or operations.
